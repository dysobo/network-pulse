# NetworkPulse - 家庭网络监控仪表盘

集中查看家庭网络关键服务状态的 Android 应用。

## 支持服务

| 服务 | 功能 |
|------|------|
| iKuai | 在线终端、CPU/内存、公网 IP、连接数 |
| AdGuard Home | 查询统计、拦截率、规则数量 |
| Mihomo | 活动连接、节点信息、国际链路延迟 |
| Lucky | Web 规则、子规则列表、证书状态 |

## 功能特性

- 内网优先、外网自动回退
- 手机网络状态实时显示（WiFi/蜂窝/信号强度）
- 国内/国际链路延迟探测
- iKuai 终端设备本地命名
- 服务详情弹窗与子规则跳转
